# WeatherPRO / ПогодаPRO

A multilingual weather application built with React + TypeScript + Vite.

## Features
- Search weather by city name with autocomplete
- 5-day forecast
- Favorites and search history (localStorage)
- RU/EN language switching with color themes
- Day/Night mode
- Interactive map (Leaflet.js)

## Setup
1. Copy `.env.example` to `.env`
2. Add your OpenWeatherMap API key to `.env`
3. Run `npm install`
4. Run `npm run dev`
